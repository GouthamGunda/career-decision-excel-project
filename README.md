# Career Decision Analysis (Excel Project)

## 📌 Overview
This project is built using Microsoft Excel to help analyze and support career decision-making based on different factors.

## 📊 Features
- Comparison of career options
- Data organization and analysis
- Use of Excel formulas for decision support
- Structured evaluation system

## 🛠 Tools Used
- Microsoft Excel

## 📂 Files Included
- Career Decision time.xlsx

## 📈 Key Functions Used
- IF()
- SUM()
- AVERAGE()
- COUNT()

## 💡 Insights
- Helps evaluate different career paths
- Supports decision-making using data
- Improves analytical thinking
